# Walkthrough - Responsive Menu Verification

デスクトップとモバイルでのメニュー表示の切り替え、およびモバイルメニューの高度な動作（オーバーレイ、アニメーション）の確認手順です。

## Changes

### `Header.astro`
- **デスクトップ表示 (>= 768px)**:
    - リボンメニューを右寄せで表示。
    - ハンバーガーボタンは非表示。
- **モバイル表示 (< 768px)**:
    - ハンバーガーメニューのUXを刷新。
    - **オーバーレイ**: メニュー展開時、画面全体に半透明の背景（`--color-bg`のopacity 0.7）を表示。
    - **レイアウト**: メニューリストは画面右端に配置し、画面下端まで伸長。左側には広い余白（オーバーレイ領域）を確保。
    - **アニメーション**: メニューは画面右外から左へスライドイン。
    - **タプ操作**: オーバーレイ部分（メニュー外）をクリックすると閉じる。

## Verification Steps

### 1. Desktop View (Window Width >= 768px)
- [ ] ヘッダーの右側（ロゴの反対側）に「Home」「About」「Blog」「Work」のリボンメニューが表示されていることを確認してください。
- [x] ハンバーガーメニュー（三本線アイコン）が表示**されていない**ことを確認してください。
- [x] メニュー項目にカーソルを合わせると、色が変化することを確認してください。
- [x] 画面右側にテーマ切り替えスイッチ（太陽/月）が表示されていることを確認してください。

### 2. Mobile View (Window Width < 768px)
- [x] リボンメニューが消え、ハンバーガーメニュー（三本線アイコン）が表示されていることを確認してください。
- [x] ハンバーガーメニューをクリックしてください。
    - [x] **オーバーレイ**: ヘッダー以下の画面全体に半透明の膜が表示されることを確認してください。
    - **アニメーション**: メニュー項目を含むパネルが、画面の**右側**からスライドインしてくることを確認してください。
    - **レイアウト**:
        - メニューパネルが画面右端に配置されていること。
        - 画面左側に広い余白（オーバーレイが見える部分）があること。
        - パネルの高さが画面下端まであること。
    - **テキスト**: メニュー項目の文字が太字で、左揃えで表示されていることを確認してください。
- [x] メニュー外（左側の暗くなった領域）をクリックしてください。
    - [x] メニューが右側へスライドアウトし、閉じることを確認してください。
- [x] 再度メニューを開き、閉じるボタン（✕）をクリックしてメニューが閉じることを確認してください。

### 3. Typography & Theme
- [x] ヘッダーの「TECH☆TECH」ロゴが太字であることを確認してください。
- [x] ダークモード/ライトモードを切り替え、オーバーレイの色が背景色（`--color-bg`）に基づいて変化し、自然に見えるか確認してください。
