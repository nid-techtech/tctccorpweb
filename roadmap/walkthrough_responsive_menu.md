# Walkthrough - Responsive Menu Verification

デスクトップとモバイルでのメニュー表示の切り替えと、スタイル改善の確認手順です。

## Changes

### `Header.astro`
- デスクトップ表示用メニュー (`.desktop-menu`) を追加しました。
- CSSメディアクエリを追加し、画面幅 768px を境界に表示を切り替えるようにしました。
    - **Desktop (>= 768px)**: リボンメニュー表示、ハンバーガーボタン非表示
    - **Mobile (< 768px)**: リボンメニュー非表示、ハンバーガーボタン表示
- モバイルメニュー（ハンバーガーメニュー展開時）のスタイルを改善しました。
    - 幅を画面の80%にし、右寄せ（左側に20%の余白）にしました。
    - メニュー項目を左揃えにし、余白を広げて押しやすくしました。
- ロゴとメニュー項目のフォントを太字 (Bold) にしました。

## Verification Steps

### 1. Desktop View (Window Width >= 768px)
- [ ] ヘッダーの右側（ロゴの反対側）に「Home」「About」「Blog」「Work」のリボンメニューが表示されていることを確認してください。
- [x] ハンバーガーメニュー（三本線アイコン）が表示**されていない**ことを確認してください。
- [x] メニュー項目にカーソルを合わせると、色が変化することを確認してください。
- [x] 画面右側にテーマ切り替えスイッチ（太陽/月）が表示されていることを確認してください。

### 2. Mobile View (Window Width < 768px)
- [x] リボンメニューが消え、ハンバーガーメニュー（三本線アイコン）が表示されていることを確認してください。
- [x] ハンバーガーメニューをクリックしてください。
    - [x] メニューが右側からではなく、左側に20%の余白を残して画面の80%の幅で表示されることを確認してください。
    - [x] メニュー項目が左揃えで表示されていることを確認してください。
    - [x] メニュー項目の文字が大きく、太字で表示されていることを確認してください。
- [ ] メニュー外（左側の余白部分など）をクリックしてもメニューが閉じるか確認してください（現状の実装では閉じない可能性がありますが、UIの確認として）。
    - *注: 今回の実装では閉じる動作のJavaScript変更は行っていませんが、既存の動作を確認してください。*
- [x] 閉じるボタン（✕）をクリックしてメニューが閉じることを確認してください。

### 3. Typography
- [x] ヘッダーの「TECH☆TECH」ロゴが太字であることを確認してください。
- [x] メニュー項目のテキストが太字であることを確認してください。
